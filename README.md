# 🎮 Rock Paper Scissors Game (C++)

## 📖 Overview
This project is a console-based implementation of the classic **Rock-Paper-Scissors** game built in C++.

The player competes against the computer over multiple rounds, with the system tracking scores and displaying the final winner.

---

## 🚀 Features
- 🎲 Random computer choices
- 🎯 Multi-round gameplay (1 to 10 rounds)
- 🧠 Automatic winner detection per round
- 📊 Score tracking (Player vs Computer vs Draw)
- 🎨 Colored console output based on winner
- 🔁 Option to replay the game

## 🏗️ System Design

### 🔹 stRoundInfo
Stores information about each round:
- Player choice
- Computer choice
- Round winner

### 🔹 stGameResults
Stores final game statistics:
- Total rounds
- Wins (Player / Computer)
- Draws
- Final winner

---

## ⚙️ How It Works

1. Player selects number of rounds (1–10).
2. In each round:
   - Player chooses (Stone, Paper, Scissors)
   - Computer generates a random choice
   - Winner is determined
3. Results are displayed after each round.
4. Final results are shown at the end.
5. Player can choose to play again.
